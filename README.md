# PACKET ANALYSIS PORTFOLIO

CAPTURE • ANALYZE • UNDERSTAND

---

## Overview

A network traffic analysis project that captures and examines packets using Wireshark to identify protocols, inspect communication flows, troubleshoot network issues, and analyze packet-level data. Includes packet captures, analysis reports, and documentation demonstrating protocol behavior and network diagnostics.

---

## Skills Demonstrated

- Wireshark  
- TCP/IP  
- HTTP  
- DNS  
- TLS/SSL  
- ICMP  
- DHCP  
- Network Troubleshooting  
- Packet Filtering  
- Protocol Analysis
