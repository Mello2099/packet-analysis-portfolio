# 📡 Packet Analysis Portfolio

> **CAPTURE • ANALYZE • UNDERSTAND**

A hands-on network traffic analysis project focused on capturing, inspecting, and analyzing packet-level data using Wireshark. This portfolio demonstrates how different network protocols behave in real traffic and how to diagnose network issues through deep packet inspection.

---

## 🚀 Overview

This project explores real-world network traffic using packet captures to:

- Analyze protocol communication flows
- Identify and interpret network behavior
- Troubleshoot connectivity issues
- Distinguish normal vs suspicious traffic
- Understand how core network protocols operate

Each investigation includes packet captures, analysis notes, and protocol breakdowns.

---

## 🧠 Skills Demonstrated
 Wireshark Packet Analysis, TCP/IP Networking, HTTP & HTTPS Analysis, DNS Query Resolution, TLS/SSL Handshakes, ICMP Diagnostics, DHCP Behavior Analysis,  Network Troubleshooting, Packet Filtering & Inspection, and Cybersecurity Fundamentals
